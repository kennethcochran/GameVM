# Tiger Game.com

## System Overview
- CPU: Sharp SM8521
- CPU Clock: 12 MHz
- Release Year: 1997
- Generation: 5th
- Region: Worldwide
- Predecessor: None
- Successor: Game.com Pocket Pro
- Notable Feature: First handheld with a touchscreen and Internet connectivity
