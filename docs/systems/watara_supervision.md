- Generation: 4th

- Predecessor: None

- Notable Feature: Game Boy competitor with TV-out capability

## CPU Details
### WDC 65C02
#### Architecture Characteristics
- Processor: 8-bit CMOS
- Word Size: 8-bit
- Endianness: Little-endian
- Register Set:
  - Accumulator (A)
  - Index Registers (X, Y)
  - Stack Pointer
  - Program Counter
  - Status Register
- Features:
  - Enhanced 6502 instruction set
  - Low power CMOS
  - Additional addressing modes
  - Bug fixes from NMOS 6502

## Memory Map
### System Memory
- Work RAM: 8KB
- Video RAM: 8KB
- ROM: 2KB (boot)
- Cartridge ROM: Up to 64KB

### Memory Layout
- $0000-$1FFF: Work RAM
- $2000-$3FFF: Video RAM
- $4000-$7FFF: Reserved
- $8000-$FFFF: Cartridge ROM
- Zero Page: System variables
