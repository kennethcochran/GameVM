# Epoch Cassette Vision

## System Overview
- CPU: NEC µPD7810
- CPU Clock: 4 MHz
- Release Year: 1981
- Generation: 2nd
- Region: Japan
- Predecessor: None
- Successor: Super Cassette Vision
- Notable Feature: First Japanese cartridge-based game console

## CPU Details
### NEC µPD7810
#### Architecture Characteristics
- Processor: 8-bit microcontroller
- Word Size: 8-bit
- Endianness: Little-endian
- Register Set:

  - Program Counter
  - Stack Pointer
  - Program Status Word
- Features:
  - Built-in timer
  - I/O ports
  - Interrupt control
  - Serial interface
