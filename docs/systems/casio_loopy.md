# Casio Loopy

## System Overview
- CPU: NEC V30MZ
- CPU Clock: 20 MHz
- GPU: Custom sprite processor
- Release Year: 1995
- Generation: 4th/5th (transitional)
- Region: Japan only
- Predecessor: None (Casio's first console)
- Successor: None
- Notable Feature: Built-in thermal printer and "My Seal" creation system
